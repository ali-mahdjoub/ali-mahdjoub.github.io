Android
Java

A complete Android mobile solution I built from scratch using Java and Firebase, designed to support the digital management of oil wells, deposits, and field locations in industrial operations.

This app empowers field operators with tools for tracking, planning, and analyzing oil field resources with real-time maps, structured PDF reports, and an intuitive dashboard.

✨ Key Features
  • Authentication & Profiles – Secure login, signup, and profile updates

  • Distance & Comparison Tool – Compare distances between wells and generate PDF reports
  • Deposits Module – Register, categorize, and manage deposits per site
  • Geolocation Module – Track live routes and destinations using Google Maps API
  • PDF Export – Generate structured comparative reports for teams
  • Dashboard UI – Clean and modern interface with user-friendly navigation
